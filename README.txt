# ReadMe for Honeyports.py

- Its pretty to use the tool.
- Open a command prompt - "sudo python honeyports.0.1.1.py" (Linux Distro)
                          "python honeyports.0.1.1.py" (Windows OS)
- Select the number of ports
- Enter the port number - **Ports from 1 - 1024 require administrative / superuser privillages **
- Once attacker(s) IP address has been captured select further choice.

-- Requirements :
     1. Python 2.7.x
     2. py2-ipaddress-2.0
     3. colorama-0.3.1
     4. termcolor-1.1.0
